# Yemek Fotoğrafları

Bu klasörde menü öğelerinin fotoğrafları saklanır.

## Klasör Yapısı

```
images/
├── breakfast/          # Kahvaltı fotoğrafları
├── soups/             # Çorba fotoğrafları
├── starters/          # Başlangıçlar & Atıştırmalıklar
├── salads/            # Salata fotoğrafları
├── phuket/            # Phuket İmza Yemekleri
├── seafood/           # Deniz Ürünleri
├── western/           # Batı Ana Yemekleri
├── comfort/           # Comfort Food
├── pasta/             # Makarnalar
└── desserts/          # Tatlılar
```

## Fotoğraf Yükleme

### Adım 1: Fotoğrafları Hazırlayın
- **Format:** JPG veya PNG
- **Boyut:** Minimum 800x600px (ideal: 1200x900px)
- **Kalite:** Yüksek çözünürlük
- **İsimlendirme:** Küçük harf, tire ile (örn: `smashed-avocado.jpg`)

## Fotoğraf Kullanımı

```

```

## Not

- Tüm fotoğraflar web için optimize edilmelidir
- Dosya boyutu 500KB'dan küçük tutulmalıdır
- Yatay (landscape) format önerilir (4:3 veya 16:9)
